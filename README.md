Mail Module
===========

version: v1.0

Install
-------

#### With composer

```json
    "require": {
        "boeve-web-development/mail" : "dev-master"
    }
```

#### Configuration

- Copy the file mail.local.php.dist to config/autoload/mail.local.php
- And fill in the required and optional options.
